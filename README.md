# Library Management System (Java + MySQL)

This is a simple menu-driven Library Management System built using Core Java, JDBC, and MySQL.

## Features
- Add Book
- Delete Book
- View All Books
- View All Books using command line menu

## Technologies Used
- Java
- JDBC
- MySQL
- VS Code

## Project Structure
librarymanagement/
- dao       → Data Access Layer
- dto       → Data Transfer Objects
- service   → Business Logic
- util      → Database Connection
- main      → Main class

## How to Run
1. Create MySQL database and table
2. Add MySQL Connector JAR
3. Compile and run using command line
